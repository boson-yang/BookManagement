# 图书管理系统
#### 基于Spring+Spring MVC(Maven方式构建)

### 项目简介
本图书管理系统基于spring，spring mvc，数据库为H2，前端使用了Bootstrap。 
### 系统功能
该系统实现读者和管理员登录，图书的增删改查，读者的增删改查，借还图书，密码修改等功能。
### 如何使用
```
$ git clone https://github.com/boson-yang/BookManagement

$ cd BookManagement

$ mvn clean compile

$ mvn clean package

$ mvn clean install

$ mvn jetty:run

http://localhost:9000
```
